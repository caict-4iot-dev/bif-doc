星火·链网开放文档
===================================

“星火·链网”是在工信部的领导与专项支持下，由 中国信通院牵头、联合北航、北邮、中国联通等 多家大型企事业单位建设的国家区块链新型融合 基础设施体系，是为持续推进产业数字化转型， 推动数字资产价值化，利用区块链自主创新能力 而谋划布局的数字经济“新型基础设施”。
